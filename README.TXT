A LAMP stack web-application for COP4331.

Allows user to create account, which will be updated and stored on backed server.
It will then allow users to create and store information for each contact they wish to keep. 
Contacts can be added with incomplete information (e.g. no work phone).
Contacts can be editied, as well as deleted.

Hosted on AWS EC2 Server using Apache Webserver and MySql.

AWS operation and dns routing completed by Christopher Beirne.
